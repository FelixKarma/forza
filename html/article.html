Patch avril 2023

Le studio Playground Games annonce un futur patch pour son jeu phare Forza Horizon 5, dont le déploiement est prévu pour le 25 avril 2023. La mise à jour prévoit, évidemment, diverses corrections de bugs, affectant certaines missions, les performances sur PC, ou certaines routes qui n'obtenaient pas le statut de découvertes.

Un nouveau patch vient d'être publié pour le jeu Forza Horizon 5, ce dernier pèse environ 2 Go et se téléchargera automatiquement par le biais du store de Microsoft ou de Steam.Le but est de corriger divers bugs, comme des soucis de passage de rapports en mode manuel, des plantages lors de l'activation de la technologie XeSS d'Intel...

Forza Horizon 5 May 1st Update Release Notes

  • Fixed an issue where gears were not working correctly for players using manual shifting
  • Fixed an issue with Horizon Open where sometimes leaderboards were not displaying results correctly
  • [PC] Fixed an issue where changing Intel XeSS settings could result in a crash

Bugs corrigés

  • Correction d’un problème où parfois les défis hebdomadaires #ForzaThon étaient réinitialisés après le redémarrage du jeu
  • Correction d’un problème où parfois les routes n’étaient pas marquées comme découvertes sur la carte
  • Correction d’un problème avec l’hôtel des ventes où la recherche par « Type de voiture » ne pouvait renvoyer aucun résultat
